# 🎞️ Projeto Básico: Blog Brazil

Este projeto é uma aplicação simples desenvolvida com **HTML5**, **CSS3** e **JavaScript**, com o objetivo de praticar os principais conceitos de usabilidade.

A proposta é explorar conceitos básicos de desenvolvimento front-end enquanto implementa usabilidade.

## Objetivo do projeto

- Praticar fundamentos de **HTML, CSS e JavaScript**
- Aprender a integrar e utilizar bibliotecas externas
- Praticar fundamentos de usabilidade

## Tecnologias utilizadas

- **HTML5** – estrutura da página  
- **CSS3** – estilização e layout responsivo  
- **JavaScript** – interações básicas  
- **Swiper.js** – biblioteca para criação de sliders
